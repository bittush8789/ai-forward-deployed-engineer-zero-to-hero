# Vector Databases for Enterprise AI Platforms (Phase 7)

Welcome to the **Vector Databases** curriculum. This curriculum focuses on managed and self-hosted vector databases, similarity search pipelines, hybrid/semantic query retrievals, embedding models performance, and production database operations.
