# Phase 6: AI Agent Engineering Academy

Welcome to the **AI Agent Engineering Academy**. This phase of the AI Forward Deployed Engineer (FDE) curriculum covers the design and deployment of autonomous systems, multi-agent frameworks (CrewAI, AutoGen), state chart orchestrators (LangGraph), memory systems, tool call sanitization, and production-grade agent platforms.

This curriculum focuses on how to build, optimize, secure, and govern reliable agentic systems.

---

## 🗺️ Course Syllabus

The academy consists of **8 theoretical modules (90%)** and **5 practical projects (10%)**, alongside an interactive dashboard for hands-on planning and calculations.

### 📚 Theory Modules
1. **[Module 1: LangChain](file:///d:/ai-forward-deployed-engineer-zero-to-hero/agent_engineering/modules/01_langchain.md)**: Prompt templates, output parsers, memory bindings, retrieval chains, and agent execution modules.
2. **[Module 2: LangGraph](file:///d:/ai-forward-deployed-engineer-zero-to-hero/agent_engineering/modules/02_langgraph.md)**: Stateful graphs, node actions, conditional routing edges, Human-in-the-loop checkpoints, and state recovery.
3. **[Module 3: CrewAI](file:///d:/ai-forward-deployed-engineer-zero-to-hero/agent_engineering/modules/03_crewai.md)**: Collaborative crews, role delegation, task configurations, and consensus protocols.
4. **[Module 4: AutoGen](file:///d:/ai-forward-deployed-engineer-zero-to-hero/agent_engineering/modules/04_autogen.md)**: Conversational agents, autonomous loops, user feedback routes, and multi-agent chats.
5. **[Module 5: Multi-Agent Systems](file:///d:/ai-forward-deployed-engineer-zero-to-hero/agent_engineering/modules/05_multi_agent_systems.md)**: Supervisor coordinators, hierarchical routing, peer-to-peer delegation, and coordination.
6. **[Module 6: Tool Calling](file:///d:/ai-forward-deployed-engineer-zero-to-hero/agent_engineering/modules/06_tool_calling.md)**: Function calling schemas, API gateway bindings, database access, and validation.
7. **[Module 7: Memory Systems](file:///d:/ai-forward-deployed-engineer-zero-to-hero/agent_engineering/modules/07_memory_systems.md)**: Memory taxonomy (short-term, long-term, semantic, episodic, vector), retrieval, and optimization.
8. **[Module 8: Agent Orchestration](file:///d:/ai-forward-deployed-engineer-zero-to-hero/agent_engineering/modules/08_agent_orchestration.md)**: Lifecycle management, state persistence, task queueing, and reliability.
