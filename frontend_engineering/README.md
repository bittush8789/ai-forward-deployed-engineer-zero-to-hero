# Frontend Engineering Academy

Welcome to the **Frontend Engineering Academy** – Phase 9 of the AI Forward Deployed Engineer roadmap. This curriculum equips you to build production‑grade, enterprise‑scale frontends for AI products, SaaS platforms, analytics dashboards, and LLM‑driven applications.

## 📚 Curriculum Overview
| Module | Title | Core Topics |
|---|---|---|
| 01 | **HTML** | Semantic markup, accessibility, SEO, enterprise layouts |
| 02 | **CSS** | Box model, Flexbox, Grid, responsive design, CSS architecture, theming |
| 03 | **JavaScript** | ES6+, async/await, DOM, event handling, API integration |
| 04 | **TypeScript** | Types, interfaces, generics, API contracts, large‑scale safety |
| 05 | **React** | Components, hooks, context, performance, production patterns |
| 06 | **Next.js** | Routing, server/client components, SSR/SSG, SEO, deployment |
| 07 | **Tailwind CSS** | Utility‑first design, design systems, theming, accessibility |
| 08 | **Streamlit** | Rapid prototyping, data visualization, deployment for AI tools |
| 09 | **Gradio** | Model demos, UI wrappers, deployment, security |
| 10 | **Frontend Architecture** | Component hierarchy, state management, routing, scalability |
| 11 | **Frontend Security** | XSS, CSRF, auth integration, token handling, compliance |
| 12 | **AI Frontend Engineering** | Chat widgets, RAG UI, agent dashboards, analytics |

## 🎯 Learning Approach
- **Theory (90 %)** – industry standards, enterprise architecture, best practices, security, accessibility, performance.
- **Practical (10 %)** – 5 capped projects that illustrate real‑world AI frontends.

## 📦 Projects
1. **Enterprise AI Chat Platform** – React + Tailwind mock UI with streaming messages.
2. **RAG Knowledge Assistant UI** – HTML/CSS/JS prototype calling a mock RAG backend.
3. **AI Agent Dashboard** – React dashboard using Zustand for state, mock WebSocket updates.
4. **AI Analytics Platform** – Streamlit app visualizing telemetry data.
5. **Enterprise AI SaaS Platform** – Next.js app with TypeScript, Tailwind, JWT auth mock.
