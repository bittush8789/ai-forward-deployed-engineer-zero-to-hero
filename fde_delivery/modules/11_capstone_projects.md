# Module 11: FDE Delivery Capstone Projects

This module outlines the requirements, stakeholder alignment, and delivery milestones for the five FDE capstone projects.

---

## Project 1: Insurance Claims Copilot Delivery

### Overview
Lead the discovery, requirements, PoC validation, and deployment of an AI claims processing copilot for an insurance client.

### FDE Delivery Milestones
1.  **Discovery Workshop**: Conduct As-Is process mapping and identify claims bottlenecks.
2.  **Requirements Document**: Draft the BRD and FRD, defining success targets and latency limits.
3.  **PoC Validation**: Build and deploy a core prototype to validate document extraction accuracy.
4.  **Production Deployment**: Execute a canary rollout and establish rollback strategies.

---

## Project 2: Banking AI Assistant Integration

### Overview
Manage stakeholder alignment, architectural design, and legacy database integrations for a retail banking assistant.

### FDE Delivery Milestones
1.  **Stakeholder Management**: Map power vs interest metrics and align security leads.
2.  **Solution Design**: Draft the Solution Design Document (SDD), detailing SSO and API integrations.
3.  **Enterprise Integration**: Securely connect the assistant to legacy Core Banking Systems (CBS) APIs.

---

## Project 3: Enterprise Knowledge Assistant Adoption

### Overview
Drive change management, user enablement, and customer success monitoring for an enterprise RAG search platform.

### FDE Delivery Milestones
1.  **Change Management**: Build change champion networks and lead user training.
2.  **Customer Success**: Track monthly active users (MAU) and compile ROI metrics.
3.  **Quarterly Business Review**: Present cost savings and performance updates to executive sponsors.

---

## Project 4: AI Agent Platform Architecture

### Overview
Design the system architecture, run PoC validations, and plan the phased rollout of a multi-agent orchestration platform.

### FDE Delivery Milestones
1.  **System Architecture**: Design orchestrators, stateful databases, and API interfaces.
2.  **PoC Validation**: Validate agent routing accuracy and tool execution latency.
3.  **Phased Rollout**: Implement a department-by-department deployment schedule.

---

## Project 5: Enterprise AI Transformation Program

### Overview
Lead a consulting engagement to design the AI adoption roadmap, governance frameworks, and change management strategies for a multinational enterprise.

### FDE Delivery Milestones
1.  **Technical Consulting**: Analyze business problems, map technology solutions, and compile strategic recommendations.
2.  **Governance Framework**: Detail data security policies, compliance regulations, and audit processes.
3.  **Change Management**: Structure the communication strategy and adoption targets across departments.
